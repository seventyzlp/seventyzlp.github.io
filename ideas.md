<text>
**设计运动**: 赛博朋克未来主义 (Cyberpunk Futurism)

**核心原则**:
1. **工业科技美学**: 借鉴明日方舟的工业风格，使用几何线条、科技感边框和机械质感元素
2. **动态粒子系统**: Three.js驱动的粒子背景，营造深空感和科技氛围
3. **深色主题主导**: 黑色/深灰背景为主，白色/青色文字，橙黄色作为强调色
4. **信息密度平衡**: 在科技感界面中保持内容的可读性和层次感

**色彩哲学**:
- **主色调**: 深空黑 (#0a0a0a) 作为背景，象征无尽的可能性
- **强调色**: 电子蓝 (#00d4ff) 和警示橙 (#ff6b35)，代表科技与活力
- **辅助色**: 灰色系 (#1a1a1a, #2a2a2a, #3a3a3a) 构建层次
- **粒子色**: 白色到透明的渐变，模拟星尘效果

**布局范式**:
- **非对称分栏布局**: 避免居中对齐，采用左右分栏或对角线构图
- **视差滚动**: 不同层次的内容以不同速度滚动，增加深度感
- **模块化卡片**: 使用带边框和阴影的卡片组织内容，每个卡片如同控制面板
- **全屏区块**: Hero区域和主要内容区采用全屏设计，强化沉浸感

**标志性元素**:
1. **粒子背景**: 向上缓缓飘动的白色粒子，使用Canvas或Three.js实现
2. **扫描线效果**: 细微的水平扫描线动画，增强屏幕感
3. **几何装饰**: 斜线切割、六边形边框、科技感线条装饰

**交互哲学**:
- **响应式反馈**: 鼠标悬停时元素发光、边框变色
- **流畅过渡**: 所有状态变化使用缓动动画，避免生硬跳转
- **粒子互动**: 鼠标移动时粒子产生微妙的避让或吸引效果
- **渐进式加载**: 内容区域从下往上淡入，模拟系统启动

**动画指南**:
- **入场动画**: 使用 fade-in + slide-up 组合，持续时间 0.6-0.8s
- **悬停动画**: 0.3s 的 ease-out 过渡，配合轻微的 scale 或 glow 效果
- **粒子动画**: 使用 requestAnimationFrame 实现 60fps 流畅动画
- **滚动动画**: Intersection Observer 触发，元素进入视口时播放

**字体系统**:
- **标题字体**: Orbitron (Google Fonts) - 几何感强的科技字体，用于大标题和导航
- **正文字体**: Inter - 现代无衬线字体，保证长文本可读性
- **代码字体**: JetBrains Mono - 用于技术技能和代码片段展示
- **字重层次**: 
  - 主标题: Orbitron Bold (700)
  - 副标题: Orbitron Medium (500)
  - 正文: Inter Regular (400)
  - 强调: Inter SemiBold (600)

**设计运动**: 新野兽派 (Neo-Brutalism)

**核心原则**:
1. **粗犷几何**: 大胆的矩形块、厚重边框、强烈对比
2. **扁平化深度**: 通过颜色块和阴影创造层次，而非渐变
3. **功能优先**: 每个元素都有明确的功能指向
4. **视觉冲击**: 使用高对比度和大字体创造视觉张力

**色彩哲学**:
- **主色**: 纯黑 (#000000) 和纯白 (#ffffff) 的极致对比
- **强调色**: 霓虹黄 (#ffff00) 和电光紫 (#9d00ff)
- **功能色**: 红色警告、绿色成功、蓝色信息
- **无渐变**: 所有颜色都是纯色块，拒绝渐变和模糊

**布局范式**:
- **网格系统**: 严格的12列网格，但打破常规对齐
- **重叠堆叠**: 元素之间有意重叠，创造层次感
- **非对称平衡**: 左重右轻或上重下轻的不平衡构图
- **留白激进**: 大面积留白与密集信息区形成对比

**标志性元素**:
1. **厚重边框**: 5-8px 的黑色边框包裹所有卡片
2. **硬阴影**: 8-12px 的纯色偏移阴影，无模糊
3. **几何图标**: 使用简单几何形状构建图标系统

**交互哲学**:
- **即时反馈**: 点击时元素位移，阴影消失
- **无过渡**: 状态变化瞬间完成，拒绝缓动
- **强烈对比**: 悬停时背景和文字颜色完全反转

**动画指南**:
- **跳跃动画**: 使用 steps() 函数创造逐帧动画效果
- **快速切换**: 所有过渡时间不超过 0.15s
- **位移交互**: 点击时元素向右下移动 4px，阴影同步消失

**字体系统**:
- **标题**: Space Grotesk Bold - 几何感强，笔画粗重
- **正文**: IBM Plex Sans - 清晰易读，带有机械感
- **装饰**: Bebas Neue - 用于大号标题和数字展示

## 最终选择：赛博朋克未来主义

我选择了**赛博朋克未来主义**设计方案，因为它最贴合明日方舟官网的工业科技美学，同时能够通过粒子系统和深色主题营造出独特的视觉氛围。这个方案在保持专业性的同时，也能展现个人作品集的科技感和创新性。

### 实施要点

1. **粒子系统优先**: 使用 Three.js 或 Canvas 实现核心粒子动画
2. **色彩一致性**: 严格遵循深空黑、电子蓝、警示橙的配色方案
3. **字体加载**: 在 index.html 中引入 Orbitron 和 Inter 字体
4. **动画性能**: 确保粒子动画在 60fps 运行，使用 requestAnimationFrame
5. **响应式适配**: 移动端减少粒子数量，保持流畅体验
